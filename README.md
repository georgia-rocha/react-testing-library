# Para clonar e testar

1 - Clone o repositório com o comando **git clone**:

```
git clone git@github.com:georgia-rocha/react-testing-library.git
```

2 - Entre no diretório que acabou de ser criado:

```
cd react-testing-library
```

3 - Para o projeto funcionar na sua máquina, será necessário instalar suas dependências, para isso, utilize o comando **npm install**:

```
npm install
```

4 - Pronto, agora o projeto está pronto para ser rodado localmente, utilizando o comando **npm test** você poderá testar a aplicação:

```
npm test
```

## O que foi desenvolvido?

* Foram criados testes utilizando Jest e a biblioteca React Testing Library em uma aplicação em React já construida.
